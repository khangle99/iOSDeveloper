# Portfolio Website - Le Duy Khang

A premium, responsive personal portfolio website for Le Duy Khang, iOS Developer. Built with persistent attention to performance and clean architecture, mirroring the professional standards of the developer.

## Features
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Dark Mode Theme**: Premium dark aesthetic with blue/teal accents.
- **Smooth Animations**: Fade-in and slide-up effects for a dynamic feel.
- **Interactive Elements**: Mobile menu, hover effects, and smooth scrolling.


## How to Run Locally
1.  Clone the repository.
2.  Open the folder in your code editor.
3.  Open `index.html` in your browser.
    - Or use a live server extension (e.g., Live Server in VS Code) for a better experience.

## Deployment to GitHub Pages
1.  Create a new repository on GitHub.
2.  Push this code to the repository.
3.  Go to **Settings** > **Pages**.
4.  Under **Build and deployment**, select **Source** as `Deploy from a branch`.
5.  Select the branch (usually `main` or `master`) and folder (`/` root).
6.  Click **Save**.
7.  Your site will be live at `https://<username>.github.io/<repo-name>/`.

## Customization
- **Profile Image**: Replace the icon in `index.html` with your photo by adding an `<img>` tag inside `.profile-img-container`.
- **Links**: Update the `href` attributes in `index.html` for your Resume/CV download link.
